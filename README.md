# AI Starter Guide

Interactive website to help people who want to use AI but don't know where to start.

Live at [matthewduke1.github.io/ai-starter-guide](https://matthewduke1.github.io/ai-starter-guide/)

## How It Works

1. You see "What is your goal?"
2. Pick from 12 categories (automate, build, write, design, video, data, code, research, agents, audio, learn, productivity)
3. Get curated tool recommendations with descriptions, pricing, difficulty level, and direct links

## Categories

| Goal | Example Tools |
|------|--------------|
| Automate repetitive tasks | Zapier, Make, n8n, OpenClaw, Bardeen |
| Build a website or app | Claude Code, Bolt.new, Cursor, v0, Lovable, Replit |
| Write content or copy | Claude, ChatGPT, Jasper, Grammarly, Notion AI |
| Create images or designs | Midjourney, DALL-E 3, Adobe Firefly, Ideogram, Stable Diffusion |
| Create videos | Runway, Sora, HeyGen, Kling, Descript |
| Analyze data | Julius AI, ChatGPT Code Interpreter, NotebookLM, Tableau AI |
| Code faster | GitHub Copilot, Claude Code, Cursor, Windsurf, Codeium |
| Research anything | Perplexity, NotebookLM, Elicit, Consensus |
| Build AI agents | Claude Agent SDK, OpenAI Assistants, LangChain, CrewAI, n8n |
| Create music or audio | Suno, ElevenLabs, Udio, Descript |
| Learn something new | NotebookLM, ChatGPT, Khanmigo, Duolingo Max |
| Boost daily productivity | Gemini, Granola, Reclaim.ai, Superhuman, Raycast |

## Setup

Static HTML -- no build step. Open `docs/index.html` or deploy via GitHub Pages.
